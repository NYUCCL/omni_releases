Optimizing Memory using Neural Information (OMNI)

Data Release 2: Forgetting curves and MRI activations from independent components analysis

Data Release 2 consists of a CSV (comma delimited) datafile containing information about the memory performance of N=190 participants memorizing 45 Lithuanian-English word translations. Each item was presented five times for study and then people were tested at a variety of delays ranging from 0 to 168 hours after the initial study session. Participants also provided a judgement of learning (JOL) on a 100 point scale after study rating the likelihood they believed they would remember the word pair at test.

This release also includes fMRI activation estimates from the study trials from a subset of the participants. The activation estimates are the single trial response (average of timepoints 4-8 post trial onset) for each of a set of independent fmri components identified using group ICA implemented using the GIFT-ICA toolbox.

The purpose of this data release is to share a relatively large dataset of human memory performance on a simple list-learning second language acquisition task. The data can be used to help build and test models of human memory.

The data is available on github here: https://github.com/NYUCCL/omni_releases/tree/master/data_release_2.
